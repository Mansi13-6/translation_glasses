# 👓 AI Translation Glasses

## 📌 Project Overview

This project is an **AI-based Translation Glasses system** that performs **real-time object detection and text translation** using computer vision and deep learning.

It is designed to help users understand text in different languages instantly through a smart wearable-like system. The patent is also registered for this project 

and seed funding is also provided by the college.

---

## 🚀 Features

* 📷 Real-time camera input
* 🔍 Text detection using OCR (EasyOCR)
* 🌍 Automatic language detection & translation
* 🧠 Object detection using YOLOv8
* 🔊 Text-to-Speech output (pyttsx3)
* 🌐 Supports multiple languages (Hindi, English, etc.)

---

## 🛠️ Tech Stack

* **Python**
* **OpenCV**
* **EasyOCR**
* **YOLOv8 (Ultralytics)**
* **Deep Translator**
* **pyttsx3 (Text-to-Speech)**

## ⚙️ Installation & Setup

1️⃣ Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/translation_glasses.git
cd translation_glasses
```

2️⃣ Create virtual environment

```bash
python -m venv .venv
source .venv/Scripts/activate   # Windows
```

3️⃣ Install dependencies

```bash
pip install opencv-python easyocr ultralytics deep-translator pyttsx3
```

4️⃣ Run the project

```bash
python demo.py
```
